## 開場定調

- 痛點：相較「找師傅」「PRO360達人網」等電話簿型平台，師虎來了在案件管理與服務保障上的差異化，目前官網沒有清楚傳達；企業福委會（B2B）目前也沒有明確的官網入口。
- 這次調整以董事長PPT內容為骨架，在既有頁面上**擴充＋微調**，不做大幅結構翻新。

## 二、首頁調整方案

頁面順序：導覽列 → 輪播Banner → **品牌宣言（新增）** → **商業模式差異化（新增）** → 五大優勢 → 五步驟詳解 → App下載 → 八大服務項目 → 步驟流程 → **企業福委會（新增）** → 客戶案例 → 合作夥伴 → Footer

### 新增①　品牌宣言
- **作用**：情感破題，先建立信任感，銜接輪播Banner與下方理性論證
- **版式**：左文右圖＋背景大字「NO MORE LUCK」（stmtSplit，已定案；排除全螢幕大圖版本避免跟輪播Banner重複、排除藍底三卡版本避免跟商業模式差異化重複）
- **文案**：
  - 標題：找師傅，不該靠運氣。
  - 子句：師虎來了不只是幫您找到師傅，而是對整個修繕服務負責——從報價、施工到保固，每一步都有人陪您。
  - 雙CTA頁內錨點：「立即下載APP」捲動到App下載區／「看看師虎哪裡不一樣」捲動到商業模式差異化區

### 新增②　商業模式差異化
- **作用**：正面回答「為什麼比較貴、差在哪」，全頁核心論證區塊
- **設計想法**：大數字開場製造反差 → 四張保障卡拆解具體內容 → 對照表一眼看懂差異
- **文案（第一屏）**：
  - 大數字：**3**成訂金／**7**成驗收滿意再付款（標籤「全台唯一」⚠️**待確認是否有依據可主張，避免過度宣稱**）
  - 四張保障卡：施工過程有人協助／保固直接找師虎來了／每一筆付款都有保障／發票由師虎來了開立（各附一句說明）
- **文案（對照表）**：「一張表看懂，在師虎叫修差在哪」

| | 師虎來了 | 一般媒合／派工平台 | 自己找師傅 |
|---|---|---|---|
| 付款方式 | 3成訂金／7成驗收滿意再付款 | 由消費者與師傅自行議定 | 通常需自行談妥或先付款 |
| 施工遇到問題 | 平台專人協助處理協調 | 平台不介入，需自行協調 | 完全自己與師傅協調 |
| 保固服務 | 直接找師虎來了 | 無平台保固機制 | 事後難以追蹤保固 |
| 交易安全 | 款項由平台管理，不需私下付款 | 現場交易，缺乏平台把關 | 現金交付，全憑信任 |
| 發票 | 由師虎來了統一開立 | 多為師傅個人開立收據 | 不一定索取或提供發票 |

  - 收尾標語：我們不是把一位師傅介紹給您，而是對整個修繕服務負責。
- ⚠️ **待會議拍板**：中間欄若感覺太針對其他平台、太侵略，可考慮拿掉，改回兩欄

### 新增③　企業福委會
- **作用**：開闢B2B專屬入口，承接福委會窗口的報名需求
- **設計想法**：三張權益卡＋合作企業清單，先說服再導流；CTA連結既有Google表單，不做站內嵌入表單
- **文案**：
  - 標題：企業福委會特約廠商
  - 副文：師虎來了是多家企業福委會指定的特約修繕廠商，企業員工享有專屬修繕服務與同樣完整的平台保障。
  - 合作企業：統一證券、崇越科技、英濟科技、中華電信職工福委會、遠傳電信
  - CTA：洽詢企業特約合作 → 連結至既有Google表單（目前dc.html按鈕未接表單網址，需補上）
- ⚠️ **待確認**：這5家企業是否已正式同意在官網公開列名（需比照案例/合作標章授權辦理）

### 既有區塊：本次不動
導覽列、輪播Banner、五大優勢、五步驟詳解、App下載、八大服務項目、步驟流程、客戶案例、合作夥伴、Footer。

## 三、關於我們調整方案

頁面順序：導覽列 → Banner → **核心人物→創辦人的話** → **核心價值（文案修改）** → 品牌故事（維持不動）→ **未來展望→我們的願景** → **公司里程碑（新增）** → Footer

### 核心人物 → 創辦人的話
- 第一人稱敘事，取代原本第三人稱職稱介紹，呼應品牌宣言的信任訴求
- 文案：「每一個家，都承載著一個家庭的生活。」＋創辦人第一人稱一段話

### 核心價值（文案修改）
- 把「不該孤軍奮戰」的抽象價值觀具體化——說明陪伴實際上是什麼服務
- 文案：「修繕，不該是一個人孤軍奮戰。」＋專案管理師陪同處理的具體說明

### 品牌故事（維持不動）
- 創辦人家庭背景（父親為木工）、結合科技與傳統修繕產業的創業初衷——**保留**，不整頁拿掉

### 未來展望 → 我們的願景
- 補上現況缺乏的AI/數位化方向，只維持方向性表述，不列具體產品或時間點
- 文案：「我們想改變的，不只是修繕。」＋AI／數位科技／企業合作願景說明

### 公司里程碑（新增）
- 用具體數字＋時間軸佐證平台規模與成長軌跡
- 內容：25,000+筆完成交易／20,000+位平台會員／2,200+位合作師傅＋2019-2028時間軸
- ⚠️ **待確認**：數字是否為最新正確數據（建議跟營運核對再對外呈現）
- ⚠️ **待確認**：2027／2028兩筆標「未來藍圖」，需確保視覺上清楚區分「已發生」vs「規劃中」
