# 折價券系統

## 概述

折價券是給客戶使用的優惠功能，客戶在叫修填寫資訊時輸入優惠碼，符合條件即可獲得固定金額的折扣。折扣可針對派遣費或工程款尾款兩種不同項目。

---

## 功能架構

### 折價券類型

折價券由後台建立時決定類型，分為兩種：

| 類型 | 折抵項目 | 後台識別 | 使用限制 |
|------|----------|----------|----------|
| 派遣費折價券 | 派遣費（上限 $300） | 黃色標籤 | 加成時段不可使用 |
| 完工款折價券 | 工程款尾款 | 無標籤 | 無特殊限制 |

折扣皆為**固定金額**，無百分比（打折）機制。

### 派遣費歸零邏輯

若派遣費折價券折扣金額等於派遣費全額（$300），媒合成功後將自動跳過支付派遣費的步驟。

---

## 後台設定欄位

| 欄位 | 說明 |
|------|------|
| 優惠碼 | 自訂字串，客戶輸入用 |
| 折扣價格 | 固定折抵金額 |
| 張數 | 可設上限或無限 |
| 開始時間 / 結束時間 | 折價券有效期間 |
| 狀態 | 有效 / 失效 |
| 限制新舊用戶 | 可限定僅新用戶或僅舊用戶使用 |
| 限制使用者 | 可指定特定用戶 |
| 限制地區 | 可指定特定地區 |

後台折價券列表提供分類統計：折價券總數、派遣費折價券總數、完工款折價券總數、失效折價券。

---

## 客戶使用流程

1. 客戶在叫修填寫資訊頁面，於「優惠碼」欄位輸入代碼
2. App 送出後，**後端即時驗證**所有條件（有效期、兌換數量、新舊用戶限制、指定用戶、地區、加成時段）
3. App 接收驗證結果並顯示：
   - 無效 → 警示訊息，阻擋建立訂單
   - 有效 → 折扣套用，允許建立訂單

---

## 訂單層級紀錄

訂單建立後，後台訂單詳情頁會記錄以下折價券資訊：

- **折價券折讓**：實際折抵金額
- **使用折價券**：使用的優惠碼
- **折價券類型**：折扣派遣費 / 折扣完工款

## 異動紀錄

### 2026 年 6 月

派遣費折價券的折扣金額由原本固定的 $201（客戶實付 $99）改為後台可自由設定，上限為 $300（派遣費全額）。折至 $0 時，支付派遣費步驟自動跳過。
