# 保固機制與售後服務流程

## 概述

本文件說明訂單完工後的保固機制，包含保固期起算方式、客戶端功能、售後服務申請流程，以及保固範圍的判斷依據。

---

## 相關角色

| 角色 | 職責 |
|------|------|
| 客戶 | 申請售後服務、提供照片或配合到場確認 |
| 客服 | 致電了解狀況、判斷是否屬保固範圍、聯繫原師傅 |
| 師傅 | 於保固期內回場維修，不得向客戶收費 |
| 系統 | 計算保固期限、推進訂單狀態 |

---

## 保固期起算

保固期從**客戶確認驗收的時刻**開始計算。訂單進入「等待完成」狀態後，系統會計算保固期限，並在某個時間點將訂單狀態推進至下一階段：

- **有保固的工項**：推進至「保固中」，並顯示保固期限
- **無保固的工項**：直接由系統處理為「訂單結束」

每個工項的保固天數依普通使用或商業使用有所不同，設定於工項目錄中。

---

## 客戶端功能（保固中）

訂單處於「保固中」狀態時，客戶可使用以下三個功能：

| 功能 | 說明 |
|------|------|
| 傳送保固書 | 將該訂單的保固書寄至客戶信箱 |
| 傳送報價單 | 將該訂單的報價單寄至客戶信箱 |
| 我要售後服務 | 向後台送出售後服務申請，客戶靜待客服致電 |

---

## 售後服務申請流程

1. **客戶申請**
   客戶點擊「我要售後服務」，後台收到申請通知。

2. **客服致電確認**
   客服人員看到申請後，主動致電客戶了解狀況細節。

3. **判斷是否屬保固範圍**
   客服透過客戶描述、照片或實際到場查看，判斷問題是否屬於保固範圍。

   - **屬於保固範圍**：聯繫原師傅回場維修，訂單維持「保固中」狀態，師傅不得向客戶收費。若需進行保固範圍外的額外施工，另開新訂單處理。
   - **不屬於保固範圍**：由客服向客戶說明原因。

---

## 保固範圍與免責條款

保固責任由師虎來了平台（飛達智能股份有限公司）承擔。保固書載明之免責條款如下，以下情形不屬於保固範圍：

1. 因人為故意、過失破壞，或不可抗力因素（天災、水災、火災、地震、颱風、雷擊、外來撞擊等）導致的損壞
2. 超出原工程內容範圍之外的區域或品項
3. 因滲漏水後間接造成其他物件的損壞（如壁紙、裝潢、傢俱、家電、衣物等）
4. 保固期間內進行其他整修或裝潢，導致承包工程範圍內區域或品項的損壞

---

## 訂單結束

「保固中」的訂單在以下情況下變為「訂單結束」：

- 保固期限到期，系統自動推進
- 後台人工手動操作狀態跳脫

> **備註**：訂單生命週期中存在「出保（狀態 90）」，但目前系統實際並未使用此狀態，訂單會直接從「保固中」推進至「訂單結束」。
