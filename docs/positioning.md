# 產品定位

## 平台概述

一個修繕媒合平台，連結有修繕需求的**客戶**與提供服務的**師傅**，以數位化方式解決台灣傳統修繕市場長期存在的資訊不透明問題。

---

## 核心痛點

### 客戶端

台灣消費者找師傅的傳統管道包括：親友介紹、Google 搜尋、住家附近的水電行。但修繕屬於低頻事件，消費者通常：

- 不熟悉市場行情
- 不了解施工內容與流程
- 缺乏評估師傅品質的能力

若遇到不良業者，常見後果包括：

- 溝通不清、報價不透明導致高額費用
- 施工品質差、完工時間一再延宕
- 極端情況：「裝潢蟑螂」——收取訂金後持續追加費用，甚至施工中途跑路，客戶求償無門

### 師傅端

修繕產業在台灣面臨結構性問題：

- 社會刻板印象：工作環境差、體力負擔重、低學歷低素養
- 年輕人不願入行，師傅供給逐年減少
- 部分師傅因市場供需失衡而有恃無恐，服務品質參差不齊

---

## 平台價值主張

### 對客戶

- **資訊透明化**：修繕過程全程數位記錄，降低糾紛風險
- **第三方把關**：透過平台管理提升修繕品質
- **客服橋接**：客服全程作為客戶與師傅的溝通橋樑
- **完工保固**：部分工項提供完工後保固，有問題可透過平台追責

### 對師傅

- 獲得合理報酬
- 透過平台流程逐步建立服務標準
- 長遠目標：提升整體修繕產業形象與社會地位

---

## 信任建立機制

| 機制 | 說明 |
|------|------|
| 名人推薦 | 如吳淡如背書，建立品牌公信力 |
| 客服服務 | 全程協助溝通，降低客戶與師傅之間的摩擦 |
| 完工保固 | 部分工項提供保固，增加客戶安心感 |
| 品牌形象 | 定位為專業修繕媒合平台，塑造可信賴的品牌 |
