---

## 目錄結構

```
design-system/
├── tokens/          — 設計 token（顏色、字型、間距）
├── components/      — 元件規格（用途、variants、規則、禁用模式）
└── patterns/        — 複合版型（頁面佈局、重複使用的 UI 組合）
```
