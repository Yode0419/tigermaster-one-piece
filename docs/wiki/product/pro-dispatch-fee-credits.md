# 免派遣費次數（Pro 會員權益）

## 概述

免派遣費次數是 Pro 會員每年可使用的免派遣費額度，非加成時段媒合成功時由系統自動套用，客戶無需額外操作。

---

## 背景與脈絡

免派遣費次數為 Pro 會員的年度福利之一，目的是降低客戶叫修的初始成本。與 Tiger Points 不同，此福利完全由系統自動處理，客戶不需主動選擇。詳細的 Pro 會員方案說明請參考 [Pro 會員（師虎 Pro）](pro-membership.md)。

## 取得方式

升級 Pro 會員後，每一會員年度自動獲得 6 次，依開通日週年重置。年度重置時不會主動通知客戶。

## 查看入口（客戶端）

客戶可於**帳號頁面**點入「免派遣費次數」查看詳情，顯示：

- 當前剩餘次數
- 下次重置日期（依開通日週年計算）
- 使用說明（含自動套用說明與未用完不累積提示）

詳情頁目前不顯示歷史使用紀錄。

## 使用方式

系統自動套用，客戶無需任何操作：

- 訂單在**非加成時段**媒合成功 → 自動扣除一次，客戶事後於帳號頁面查看次數變化
- 下單過程中不會出現「即將使用一次」的提示

## 加成時段的處理

客戶填寫叫修資訊選擇加成時段（如假日全天）時，系統跳出警告 dialog：

> 注意！您所選的時段為加成時段
> 我們將會收取時間加成費用
> ※ Pro 會員免派遣費不適用於加成時段

客戶可選擇返回修改時段，或確認並繼續。

## 次數規則

- 當年度未用完的次數，年度屆滿自動失效，不累積至下一年度
- 師傅出發後若因客戶因素（現場無人、臨時取消等）導致服務中斷，仍計為使用一次

---

## 後台管理能力

客戶詳情頁的「Pro 會員」tab 顯示 Pro 會員資格狀態、開通日期，以及免派遣費次數的當前餘額與重置日期。

**免派遣費紀錄表**列出每筆異動：

| 欄位 | 說明 |
|------|------|
| 項目 | 異動類型，如「媒合成功使用」、「手動更改」 |
| 訂單 id | 關聯訂單（可點擊跳轉）；手動更改無訂單 |
| 次數增減 | 正值（+）為增加，負值（−）為扣除 |
| 剩餘次數 | 異動後的餘額 |
| 建立者 | 操作者帳號 id |
| 建立日期 | 異動時間 |

右上角「更改免派遣費次數」按鈕可手動增減次數。另有「取消 Pro 會員資格」按鈕。
