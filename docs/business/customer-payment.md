# 客戶費用總覽

## 概述

本文件說明客戶在整個修繕過程中可能支付的所有費用，包含派遣費與施工款的費率規則、支付時機、抵扣方式，以及常見的客戶反應。

---

## 一、派遣費

### 性質

派遣費是客戶叫修後、師傅承接時所收取的費用，性質類似車馬費，用以保障師傅的出工補償（含時間、車資、停車費等），無論後續施工是否成立皆屬師傅應得的基本費用。

### 費率

依叫修時段分為四級：

| 時段               | 說明                             | 費用   |
| ------------------ | -------------------------------- | ------ |
| 一般時段           | 非下列加成時段                   | $300   |
| 夜間加成           | 周一至周五 PM6:00–PM8:59         | $2,500 |
| 假日加成           | 周休二日 AM8:00–PM8:59           | $2,500 |
| 國定假日／連假加成 | 國定假日及連續假日 AM8:00–PM8:59 | $3,500 |
| 特殊時段加成       | 非上述時段（如深夜、清晨）       | $4,500 |

設計原則：將師傅視為有上下班時間的工作者，休息時段出動須有額外補償，建議客戶非緊急情況盡量在一般時段叫修。

### 支付規則

- **時機**：師傅承接後，客戶須在 30 分鐘內完成支付
- **方式**：限信用卡，不接受現金

### 抵扣規則

派遣費可全額抵扣工程款。若施工費用低於派遣費用，客戶無需補付尾款，但差額**不退款**。

> 範例一：派遣費 $300，工程款 $1,000 → 完工後尾款 $700
> 範例二：假日派遣費 $2,500，工程款 $1,000 → 完工後無需支付尾款，但 $1,500 差額不退款

### 退款規則

派遣費原則上**不退款**，以下為各情境說明：

| 情境                             | 處理方式                                           |
| -------------------------------- | -------------------------------------------------- |
| 師傅未配合（如爽約、取消）       | 派遣費保留最多一個月，可用於重新媒合，無需再次支付 |
| 一個月內未重新叫修               | 派遣費消失，不退款（TBD，實務案例極少）            |
| 師傅完成場勘報價後，客戶決定不修 | 派遣費不退款                                       |
| 客戶主動取消（未進入場勘）       | 訂單取消，派遣費不退款                             |

### 客戶感受

- 大多數客戶能理解派遣費為車馬費性質
- 在尚未接受任何服務前即需付費，心理門檻較高
- 客服致電說明時，部分客戶感覺像被催款
- 客服通常說明派遣費可抵工程款，有助於緩解疑慮

## 二、施工款

### 費用組成

施工款由**師傅報價**加上稅金與平台利潤組成，最終金額以系統計算為準。報價計算規則與加成邏輯詳見 [quotation-flow.md](../process/quotation-flow.md)。

### 支付結構

| 條件                   | 訂金           | 尾款               |
| ---------------------- | -------------- | ------------------ |
| 施工費用未超過 $50,000 | 無訂金         | 完工驗收後一次付清 |
| 施工費用超過 $50,000   | 完工前先付三成 | 完工驗收後付清餘款 |

### 支付方式

訂金與尾款均可選擇：

- 信用卡線上刷卡
- ATM 匯款
