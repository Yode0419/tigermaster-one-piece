# 發票機制

## 概述

本文件說明師虎來了平台的電子發票開立規則，涵蓋客戶收取發票（平台開給客戶）與師傅開立發票（師傅開給平台）兩側。

---

## 客戶側：平台開立發票給客戶

### 開立時機

客戶每筆付款成功後，系統自動開立電子發票，不需客戶主動申請。適用付款類型包括：

- 派遣費
- 訂金
- 尾款
- 小費

每筆付款各自獨立開立一張，不合開。

### 發票類型

客戶可在 app 個人帳號預設常用發票設定，叫修時自動套用；每次付款前仍可調整。支援以下五種類型：

| 類型 | 說明 |
|------|------|
| 會員載具 | 歸戶至師虎來了會員帳號 |
| 手機條碼載具 | 歸戶至財政部手機條碼 |
| 自然人憑證載具 | 歸戶至自然人憑證 |
| 公司發票 | 填寫抬頭與統一編號 |
| 捐贈發票 | 捐贈至指定公益單位 |

### 通知方式

平台串接 **ezpay** 電子發票服務，發票開立後由 ezpay 寄送開立通知至客戶 Email，發票號碼同步上傳至財政部平台供查詢。

### 中獎通知

- **會員載具**：由師虎來了寄送中獎通知 Email
- **手機條碼 / 自然人憑證載具**：由財政部電子發票整合服務平台通知領獎

### 作廢與補開規則

| 情境 | 處理方式 |
|------|---------|
| 發票開立後想更改載具類型 | 不受理，發票一經開立無法更改 |
| 公司抬頭或統編填錯 | 聯繫客服，原始發票作廢後重新開立，酌收手續費 **105 元（含稅）**，處理時間 **10 個工作天** |
| 捐贈發票後反悔 | 不受理，捐贈完成後所有權即屬受贈單位，亦不通知是否中獎 |

---

## 師傅側：師傅開立發票給平台

### 有成立公司的師傅

- 需開立發票給師虎來了，發票金額為**師傅報價全額**（含 5% 營業稅），非扣除佣金後的淨額
- 寄送發票時須附：掛號編號、空白處鉛筆簽名 + 路名
- 平台**收到發票後**才進行撥款

### 個人接案師傅（未成立公司）

- 不需開立發票
- 單次給付金額超過 **2 萬元**，平台依法扣繳：
  - 所得稅 **10%**
  - 二代健保費 **2.11%**
- 平台允許師傅申請**拆單**，將單筆金額拆為兩筆各低於 2 萬元，以迴避扣繳門檻（例：3 萬元拆為 19,999 + 10,001）
